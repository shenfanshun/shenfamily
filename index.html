📖 如何新增家族照片

1

建立照片資料夾

在網站根目錄建立一個 photos 資料夾。建議依年代或家族成員分成子資料夾，例如：
photos/1950s/、photos/1970s/、photos/family/

2

編輯 photos.js 設定照片資料

打開 photos.js 檔案，依照格式新增每張照片的資訊：

{ src: "photos/1960s/grandma.jpg", title: "阿嬤的青春", year: "1965", era: "1960年代", desc: "攝於台南老家門口" }

3

上傳至 GitHub

將所有檔案推送到你的 GitHub Repository，然後在 Repository 設定中開啟 GitHub Pages（Settings → Pages → Deploy from branch → main）。幾分鐘後網站就會上線。

4

照片格式建議

建議使用 JPG 格式，單張不超過 2MB，寬度建議 1200px 以上。老照片掃描解析度建議 300 DPI。GitHub Pages 免費空間為 1GB，足夠存放數百張高畫質照片。
